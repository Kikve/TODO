# 📑text input
<div class="hash">(5ff51)</div>

__type="text"__ When the type [[📑attributes (h&c)|attribute]] has a value of text, it creates a single line text input.

[[📑name (h&c)|name]]
[[📑maxlength (h&c)|maxlength]]

```html
<form action="http://www.example.com/login.php">
    <p>Username:
        <input type="text" name="username" size="15" maxlength="30" />
    </p>
</form>
```
