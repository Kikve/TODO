#nota/📑


> [!NOTE] 
>[[📑dictionaries (pytt)|dictionary]]  The key and corresponding value can be retrieved at the same time using the __items()__ method.


```python
knights = {'gallahad': 'the pure', 'robin': 'the brave'}

for k, v in knights.items():

    print(k, v)

gallahad the pure
robin the brave
```
