# Range


```python
for i in range(5):
    print(i)
```

```python
list(range(5, 10))

list(range(0, 10, 3))

list(range(-10, -100, -30))
```

```python
a = ['Mary', 'had', 'a', 'little', 'lamb']
for i in range(len(a)):
    print(i, a[i])
```
