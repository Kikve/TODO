You may put any number of case labels inside the block opened by [[ğŸ“‘switch (eloj)|switch]]. The program will start executing at the label that corresponds to the value

```javascript
switch (prompt("What is the weather like?")) {
case "rainy":
console.log("Remember to bring an umbrella.");
break;
case "sunny":
console.log("Dress lightly.");
case "cloudy":
console.log("Go outside.");
break;
default:
console.log("Unknown weather type!");
break;
}
```
